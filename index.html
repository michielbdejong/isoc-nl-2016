Internet Storage Sync

Michiel de Jong, @michielbdejong, michiel@mozilla.com

Lighting talk for Dutch Internet Society.

Abstract: Internet users want to sync data between devices and hosted servers.

The Internet we knew

  • General-purpose computers
  • … inter-connected as equal peers

The Internet we have

  • Pre-configured devices
  • … that sync with storage servers

Storage silos!

  • Windows users get One-drive
  • Android users get GoogleDrive
  • iPhone users get assigned to iCloud

The Internet we want

  • Any device manufacturer
  • Any storage provider

How?

  • Standards!
  • or…
  • Polyglot sync clients

IETF

  • Internet Storage Sync

W3C

  • Decentralized Sharing Community Group

Browser vendors

  • storage.sync
  • Web apps and browser addons can store their data
  • The browser syncs it
  • In Chrome: to Google storage
  • In Firefox (WiP): supporting multiple sync APIs
